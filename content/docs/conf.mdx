---
title: Configuration
description: Configuration options for DiceDB.
---

DiceDB extends [Valkey](https://valkey.io) and hence it supports all the configurations from [valkey.conf](https://valkey.io/topics/valkey.conf/).

Please check the [version compatibility](/versions/1.0.0) to understand which Valkey version a given DiceDB version maps to.

In addition to the standard Valkey configurations, DiceDB also supports module-specific configurations.
For details on configuring the Spill module for multi-tiering support, please see the [Multi Tiering documentation](/docs/features/tiering#configuration).
