# DiceDB Docs

This repository hosts the source code for the official
DiceDB documentation available at [dicedb.io](https://dicedb.io).

## Overview

DiceDB is an open-source, fast, reactive, in-memory
database built for modern workloads. This repository contains
all content, configuration, and assets used to build and
publish the documentation site.

## Local Development

To run the docs server locally

```bash
git clone https://github.com/dicedb/docs.git
npm install
npm run dev
```
